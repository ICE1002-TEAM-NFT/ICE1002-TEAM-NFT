# INHA 70+ 

<a href="35.199.162.143">INHA 70+ ⇗</a> : 35.199.162.143에 접속하여 저희 서비스를 확인 가능합니다.<br>

ESP32의 IR센서가 빈 강의실 내부의 이용 정보를 수집하고 이를 서버에 보내주면 서버가 웹에 이용 정보를 보여주는 방식으로 구동되며,<br>
사용자는 이를 확인하고 빈 강의실을 바로 찾을 수 있도록 하였습니다.<br>
